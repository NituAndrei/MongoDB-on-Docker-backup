2.: Start the mongo container with the command: docker run -d -p 27017-27019:27017-27019 --name mongodb mongo:4.0.9
?.: In powershell, execute: python -m pip install pymongo
?.: Populate mongo data by running DBPopulate.py (powershell command: python DBPopulate.py)
